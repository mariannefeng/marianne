#### what does it do

search for board game rulebooks by game then, then find relevant sections by searching for keywords

#### why

easier to share rulebooks while playing board games

#### how

a PWA with a <a href="https://github.com/mariannefeng/rulebook" target="_blank" rel="noopener noreferrer">golang backend and react frontend</a>. Board game PDFs are retrieved from https://en.1jour-1jeu.com/rules and stored in R2 if retrieving for the first time
